# Rift XO

Онлайн-арена крестиков-ноликов с комнатами по коду, ботом и тремя режимами:

- `Классика`: обычное поле 3x3.
- `Рифт`: у каждого игрока остается только 3 активных знака, энергетические клетки заряжают импульс.
- `Сверхнова`: поле 4x4, 4 в ряд, лимит 4 знака и периодическая вспышка, которая стирает самый старый знак.

## Запуск

```bash
npm start
```

Откройте `http://localhost:3000`.

Онлайн-комната работает через ссылку с кодом комнаты. Состояние хранится в памяти процесса сервера.
